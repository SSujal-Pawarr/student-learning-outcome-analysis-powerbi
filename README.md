# Student Learning Outcome Analysis (Power BI)

## 📊 Project Overview
This project analyzes student learning performance across Indian states and districts using the National Achievement Survey (NAS) dataset from NDAP.

## 🎯 Objectives
- Identify best performing states
- Identify difficult subjects
- Detect low performing districts
- Analyze student survey participation

## 🛠 Tools Used
- Power BI
- Power Query
- NDAP Government Dataset

## 📈 Key Insights
- Rajasthan has highest learning score
- Mathematics is most difficult subject
- Some districts show low performance
- Uttar Pradesh has highest student participation

## 📂 Files Included
- Power BI Dashboard (.pbix)
- Dataset (.csv)
- Project Presentation (.ppt)

## 🔗 Dataset Source
https://ndap.niti.gov.in
